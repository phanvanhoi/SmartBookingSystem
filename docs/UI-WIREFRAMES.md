## 1. DESIGN SYSTEM

### 1.1 Color Palette

```
── DARK THEME (Primary) ──────────────────────────────

Background:
  --bg-primary:     #0a0a0f     ← Nền chính (gần đen)
  --bg-secondary:   #111118     ← Nền card/panel
  --bg-tertiary:    #1a1a24     ← Nền hover/active
  --bg-elevated:    #222230     ← Nền modal/dropdown

Border:
  --border-default: #2a2a3a     ← Viền mặc định
  --border-hover:   #3a3a4a     ← Viền hover

Text:
  --text-primary:   #f0f0f5     ← Chữ chính (trắng nhẹ)
  --text-secondary: #8888a0     ← Chữ phụ
  --text-muted:     #555568     ← Chữ mờ

Accent (Gradient Neon):
  --accent-primary: #6c5ce7     ← Tím chủ đạo
  --accent-secondary:#a855f7    ← Tím sáng
  --accent-gradient: linear-gradient(135deg, #6c5ce7, #a855f7)

Status Colors:
  --status-available:  #22c55e  ← Xanh lá - Trống
  --status-occupied:   #ef4444  ← Đỏ - Đang hát
  --status-ending:     #f59e0b  ← Vàng - Sắp trống
  --status-maintenance:#6b7280  ← Xám - Bảo trì

Semantic:
  --success:  #22c55e
  --warning:  #f59e0b
  --error:    #ef4444
  --info:     #3b82f6
```

### 1.2 Typography

```
Font Family:  'Inter', 'Segoe UI', system-ui, sans-serif
Font Vietnamese: Hỗ trợ đầy đủ dấu tiếng Việt

Sizes:
  --text-xs:    12px    ← Labels nhỏ
  --text-sm:    14px    ← Body text
  --text-base:  16px    ← Body default
  --text-lg:    18px    ← Subheading
  --text-xl:    20px    ← Card title
  --text-2xl:   24px    ← Page title
  --text-3xl:   30px    ← Dashboard numbers
  --text-4xl:   36px    ← Countdown timer
  --text-5xl:   48px    ← Hero numbers

Weight:
  --font-normal:   400
  --font-medium:   500
  --font-semibold: 600
  --font-bold:     700
```

### 1.3 Spacing & Radius

```
Spacing (based on 4px):
  --space-1:  4px
  --space-2:  8px
  --space-3:  12px
  --space-4:  16px
  --space-5:  20px
  --space-6:  24px
  --space-8:  32px

Border Radius:
  --radius-sm:  6px     ← Buttons, inputs
  --radius-md:  8px     ← Cards
  --radius-lg:  12px    ← Modals
  --radius-xl:  16px    ← Large panels
  --radius-full: 9999px ← Badges, avatars
```

### 1.4 Shadow & Effects

```
  --shadow-sm:   0 1px 2px rgba(0,0,0,0.3)
  --shadow-md:   0 4px 6px rgba(0,0,0,0.4)
  --shadow-lg:   0 10px 15px rgba(0,0,0,0.5)
  --shadow-glow: 0 0 20px rgba(108,92,231,0.3)    ← Glow effect cho accent

Glassmorphism (cho modals):
  background: rgba(17,17,24, 0.8)
  backdrop-filter: blur(12px)
  border: 1px solid rgba(255,255,255,0.05)
```

### 1.5 Animation

```
  --transition-fast:   150ms ease
  --transition-normal: 250ms ease
  --transition-slow:   350ms ease

Micro-interactions:
  - Button hover: scale(1.02) + brighten
  - Card hover: translateY(-2px) + shadow-lg
  - Status change: pulse animation 2s
  - Countdown: color transition smooth khi < 15 phút
  - Toast: slide in from right
```

## 18. INTERACTION FLOWS

### 18.1 Check-in Flow (3 clicks)

```
[1] Click phòng trống     →  [2] Nhập tên khách (+ SĐT)  →  [3] Click "Nhận khách"
    hoặc nút "+ Nhận"         (tự động lookup khách quen)       ✅ Done!
```

### 18.2 Order Flow (3 clicks)

```
[1] Click phòng đang hát  →  [2] Click "Order thêm"  →  [3] Chọn món + số lượng
    hoặc từ phòng detail       mở menu grid                 Click "Xác nhận"
                                                             ✅ Done!
```

### 18.3 Checkout Flow (4 clicks)

```
[1] Click phòng   →  [2] Click "Checkout"  →  [3] Review bill    →  [4] Chọn PTTT
    "Sắp trống"       Hệ thống tính giá        Áp giảm giá (opt)    Xác nhận
                                                                      ✅ Done!
```

### 18.4 Keyboard Shortcuts

```
  Ctrl + K        → Mở tìm kiếm toàn cục
  Ctrl + N        → Mở dialog nhận khách nhanh
  Ctrl + O        → Mở dialog order nhanh
  Ctrl + 1..0     → Chuyển nhanh đến phòng 1-10
  Escape          → Đóng dialog/panel
  F5              → Refresh data
```
